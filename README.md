# 👤 API de Usuarios — Node.js

API REST desarrollada con Node.js para la gestión de usuarios.

Permite crear, consultar, actualizar y eliminar usuarios, además de manejar autenticación básica y estructura escalable para futuros módulos.

---

## 🚀 Funcionalidades

- Crear usuarios
- Obtener usuarios
- Obtener usuario por ID
- Actualizar usuarios
- Eliminar usuarios
- Estructura preparada para autenticación y validaciones

---

## 🛠️ Tecnologías

- Node.js
- Express
- JavaScript
- REST API
